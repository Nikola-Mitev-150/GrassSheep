In English:

Purpose: Creating a simple "desktop pet" that moves atop the taskbar.

Additional info: The program uses random numbers and a random timer to choose from 3 options - walk left, walk right, or stay in place and do an idle animation (its direction depends on the sheep's last facing direction).

Possible future additions:

 - More animations/new movements

- Interactions with the user's mouse (following it, reacting to clicks)

- Ways for the user to customize/change options

ON FIRST RUN THE PROGRAM WILL NEED A BIT MORE TIME TO GET THE FILES READY (a few seconds). ON FOLLOWING RUNS THE PROGRAM WILL ACTIVATE FASTER.

На Български:

Цел: Създаване на "екранен домашен любимец", който се движи над таскбара.

Допълнителна информация: Програмата използва случайни числа и случаен таймер, за да избира и сменя между 3 режима - ходене наляво, ходене надясно и стоене на едно място с анимация (посоката на гледане зависи от последната посока на движение).

Възможни бъдещи подобрения:

- Още анимации/нови движения

- Взаимодействия с мишката на потребителя (следване на нея, реакция при клик)

- Начини потребителят да персонализира/смени опциите

НА ПЪРВО ПУСКАНЕ ПРОГРАМАТА ЩЕ ИМА НУЖДА ОТ МАЛКО ПОВЕЧЕ ВРЕМЕ, ЗА ДА ПРИГОТВИ ИЗОБРАЖЕНИЯТА (няколко секунди). НА СЛЕДВАЩИ ПУСКАНИЯ ПРОГРАМАТА ЩЕ ТРЪГВА ПО-БЪРЗО.
